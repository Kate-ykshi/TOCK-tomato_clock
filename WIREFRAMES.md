# 番茄钟 TOCK 线框图讨论稿

版本：v0.7
日期：2026-07-10
状态：讨论中

## 1. 线框目标

本阶段目标不是做最终 UI，而是确认每个页面的结构、信息优先级和主要交互路径。

优先讨论顺序：
- 专注页
- 统计页
- 任务页
- 菜单栏状态

## 2. 全局布局

### 2.1 桌面窗口结构

整体采用左侧导航 + 右侧内容区：
主窗口默认宽度为 800px。

```text
┌──────────────────────────────────────────────────────────────┐
│  头像 今日已专注 2.5h  │                                  │
│                         │                                  │
│  专注                   │          当前页面内容             │
│  统计                   │                                  │
│  任务                   │                                  │
│                         │                                  │
└──────────────────────────────────────────────────────────────┘
```

### 2.2 左侧导航

左侧导航包含：
- 头像与今日已专注时长
- 专注
- 统计
- 任务

点击头像区域进入个人菜单，设置入口放在该菜单中。

## 3. 专注页

### 3.1 默认状态

专注页打开后，用户应第一眼看到大计时器和开始入口。

```text
┌──────────────────────────────────────────────────────────────┐
│                                                              │
│                         25:00                                │
│                    倒计时        正计时                       │
│                                                              │
│                    [ 任务输入框 ]                             │
│                    [ 选择分类  颜色 ]                         │
│                   专注 25 分    休息 5 分                    │
│                                                              │
│                         [ 开始 ]                              │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

### 3.2 开始前区域

开始前在同一区域完成：
- 搜索或输入任务
- 新建任务
- 选择倒计时或正计时
- 选择或确认任务分类
- 直接修改专注时长或休息时长
- 开始专注

任务输入框需要同时承担：
- 搜索已有任务
- 直接输入新任务
- 创建新任务时选择或新建类别
- 选择是否加入任务列表

分类选择规则：
- 如果用户选择已有任务，分类自动改为该任务已有分类
- 如果用户创建新任务，需要在任务输入框下方选择分类或新建分类
- 每个分类后面跟随一个对应的颜色选择

```text

│                                                              │
```


### 3.5 到点后正计时状态

```text
┌──────────────────────────────────────────────────────────────┐
│                                                              │
│                        +03:18                                │
│                    当前任务：写 PRD                           │
│                                                              │
│                    [ 去休息 ]  [ 结束 ]                       │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

到点后：
- 不自动进入休息
- 界面颜色轻微加深
- 倒计时切换为正计时
- 用户可选择去休息或结束

### 3.6 休息状态

```text
┌──────────────────────────────────────────────────────────────┐
│                                                              │
│                         05:00                                │
│                          休息中                              │
│                                                              │
│                         [ 结束 ]                              │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

休息状态沿用同一个大计时器，只改变文案和颜色。

## 4. 本轮需要确认

### 4.1 已确认：专注页信息量

大计时器附近不显示模式信息。

### 4.2 已确认：时长编辑方式

不再提供“模式”选择。专注时长和休息时长以数字形式直接展示，默认分别为 25 分钟和 5 分钟，用户可点击数字直接修改。

### 4.6 已确认：计时方向

大计时器下方可选择倒计时或正计时。

### 4.7 已确认：任务分类选择

搜索或输入任务下方增加分类选择。已有任务自动带出已有分类；新建任务需要选择分类或新建分类；每个分类后面跟随颜色选择。

### 4.3 已确认：休息中按钮

休息中只保留“结束”按钮。

### 4.4 已确认：正计时显示方式

到点后的正计时显示为“+03:18”。

### 4.5 已确认：开始按钮位置

开始按钮保持在任务输入框下方居中。

- 页面标题
- 编辑入口

任务页内容区按分类分组展示，所有分类默认展开。

## 8. 菜单栏状态

### 8.1 状态类型

菜单栏包含三种关键状态：
- 显示时间：默认显示当前计时，例如 24:12
- 隐藏时间：隐藏计时，仅显示 Logo 或简洁图标
- 展开菜单：点击菜单栏后展示当前任务和常用控制

### 8.2 展开菜单信息

展开菜单包含：
- 当前任务
- 当前计时
- 暂停或继续
- 结束
- 显示或隐藏时间
- 打开主窗口
